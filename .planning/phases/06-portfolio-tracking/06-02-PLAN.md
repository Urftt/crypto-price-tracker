5. `cmd_portfolio_lots(args: argparse.Namespace) -> None` — Call `get_holdings_by_symbol(args.symbol)`. If empty, print `f"No holdings found for {args.symbol.upper()}"`. Otherwise render using `render_portfolio_lots(args.symbol.upper(), lots)` (defined next task).

6. `cmd_portfolio_export(args: argparse.Namespace) -> None` — Get all holdings. If `args.export_format == "csv"`, call `export_csv(holdings)`. If `"json"`, call `export_json(holdings)`. Print result to stdout. If `args.output` is set, write to file instead and print confirmation. Note: use `args.export_format` (not `args.format`) because the argparse argument uses `dest="export_format"`.

**Pydantic models (define before `create_app`):**
```python
class HoldingCreate(BaseModel):
    symbol: str
    amount: float = Field(gt=0)
    buy_price: float = Field(gt=0)
    buy_date: str | None = None

class HoldingUpdate(BaseModel):
    amount: float | None = Field(default=None, gt=0)
    buy_price: float | None = Field(default=None, gt=0)
    buy_date: str | None = None
```

**Add these routes inside `create_app()`, after the existing `index()` route but before the static mount:**

1. `@app.get("/api/portfolio")` — `def api_portfolio():` — Get all holdings. Fetch current prices `get_top_coins(top_n=100)`, build `prices = {c.symbol: c for c in coins}`. Call `aggregate_portfolio(holdings, prices)`. Return dict with:
   ```python
   {
       "rows": [dataclasses.asdict(r) for r in summary.rows],
       "total_value": summary.total_value,
       "total_cost": summary.total_cost,
       "total_pnl_eur": summary.total_pnl_eur,
       "total_pnl_pct": summary.total_pnl_pct,
   }
   ```
   Wrap `get_top_coins` in try/except — if API fails, aggregate with empty prices dict.

2. `@app.get("/api/portfolio/lots/{symbol}")` — `def api_portfolio_lots(symbol: str):` — Return `[dataclasses.asdict(h) for h in db_get_holdings_by_symbol(symbol)]`.

3. `@app.post("/api/portfolio", status_code=201)` — `def api_add_holding(holding: HoldingCreate):` — Call `db_add_holding(holding.symbol, holding.amount, holding.buy_price, holding.buy_date)`. Return `{"id": hid, "status": "created"}`. Catch `ValueError` and raise `HTTPException(400, detail=str(e))`.

4. `@app.put("/api/portfolio/{holding_id}")` — `def api_update_holding(holding_id: int, update: HoldingUpdate):` — Build kwargs from non-None fields of `update`. Call `db_update_holding(holding_id, **kwargs)`. If returns False, raise `HTTPException(404, detail="Holding not found")`. Return `{"status": "updated"}`.

5. `@app.delete("/api/portfolio/{holding_id}")` — `def api_remove_holding(holding_id: int):` — Call `db_remove_holding(holding_id)`. If returns False, raise `HTTPException(404, detail="Holding not found")`. Return `{"status": "deleted"}`.
