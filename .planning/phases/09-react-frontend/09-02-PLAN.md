<!-- Key types and API contracts from Plan 01 that this plan builds on. -->

API endpoints (unchanged from existing backend):
- GET /api/portfolio -> { rows: AggRow[], total_value, total_cost, total_pnl_eur, total_pnl_pct }
  Each AggRow: { symbol, total_amount, avg_buy_price, current_price, current_value, total_cost, pnl_eur, pnl_pct, allocation_pct, num_lots }
- GET /api/portfolio/lots/{symbol} -> Holding[]
  Each Holding: { id, symbol, amount, buy_price, buy_date }
- POST /api/portfolio { symbol, amount, buy_price, buy_date? } -> { id, status: "created" }
- PUT /api/portfolio/{id} { amount?, buy_price?, buy_date? } -> { status: "updated" }
- DELETE /api/portfolio/{id} -> { status: "deleted" }

- GET /api/alerts -> PriceAlert[]
  Each PriceAlert: { id, symbol, target_price, direction, status, created_at, triggered_at }
- POST /api/alerts { symbol, target_price, direction } -> { id, status: "created" }
- DELETE /api/alerts/{id} -> { status: "deleted" }
- DELETE /api/alerts/triggered -> { status: "cleared", count }

- GET /api/prices -> { coins: CoinData[], triggered_alerts: PriceAlert[] }

SSE data (from /api/prices/stream):
- event: "prices"
- data: { coins: CoinData[], triggered_alerts: PriceAlert[] }
