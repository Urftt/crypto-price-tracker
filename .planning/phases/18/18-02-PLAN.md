## Task 2: Replace PriceChart hardcoded hex colors with CSS custom properties

**File(s):** `frontend/src/components/PriceChart.jsx`
**What:** Replace all 6 hardcoded hex color values with runtime reads from CSS custom properties using `getComputedStyle()`, wrapped in `useMemo` for performance.
**Depends on:** none

### Changes

**Add `useMemo` to imports (line 1):**

Change: `import { useState, useEffect } from 'react';`
To: `import { useState, useEffect, useMemo } from 'react';`

**Add CSS color resolver helper and `useMemo` call inside the component, before the `data` variable (after line 11):**

```jsx
const colors = useMemo(() => {
  const get = (prop) =>
    getComputedStyle(document.documentElement).getPropertyValue(prop).trim();
  return {
    border: get('--color-border'),         // was #21262d
    textMuted: get('--color-text-muted'),   // was #8b949e
    card: get('--color-card'),              // was #161b22
    borderLight: get('--color-border-light'), // was #30363d
    text: get('--color-text'),              // was #c9d1d9
    accent: get('--color-accent'),          // was #58a6ff
  };
}, []);
```

**Replace all hardcoded hex values in the JSX (lines 66-98):**

1. `CartesianGrid` (line 66):
   - Before: `stroke="#21262d"`
   - After: `stroke={colors.border}`

2. `XAxis` (lines 68-70):
   - Before: `tick={{ fill: '#8b949e', fontSize: 11 }}` and `stroke="#21262d"`
   - After: `tick={{ fill: colors.textMuted, fontSize: 11 }}` and `stroke={colors.border}`

3. `YAxis` (lines 72-76):
   - Before: `tick={{ fill: '#8b949e', fontSize: 11 }}` and `stroke="#21262d"`
   - After: `tick={{ fill: colors.textMuted, fontSize: 11 }}` and `stroke={colors.border}`

4. `Tooltip` `contentStyle` (lines 78-83):
   - Before: `background: '#161b22'`, `border: '1px solid #30363d'`, `color: '#c9d1d9'`
   - After: `background: colors.card`, `border: \`1px solid ${colors.borderLight}\``, `color: colors.text`

5. `Line` (lines 91-98):
   - Before: `stroke="#58a6ff"` and `activeDot={{ r: 4, fill: '#58a6ff' }}`
   - After: `stroke={colors.accent}` and `activeDot={{ r: 4, fill: colors.accent }}`

After all replacements, zero hardcoded hex color strings should remain in PriceChart.jsx. The chart should render identically since the CSS custom property values resolve to the same hex values.
